### 🛒 [AI Semantic Search E-commerce](https://github.com/PedroGoulartbranco/ai-semantic-search-ecommerce)
Sistema de busca inteligente que utiliza **Processamento de Linguagem Natural (NLP)** para compreender a intenção do usuário. Diferente de buscas comuns por palavras-chave, este projeto utiliza *embeddings* para encontrar produtos por similaridade de contexto.
- **Tech Stack:** Python, Sentence-Transformers, Vector Search, Pandas.
- **Key Feature:** Transformação de descrições textuais em vetores matemáticos para resultados de alta precisão.
